# skills 目录说明

存放可复用的操作模板（技能），供 AI Agent 在特定场景下按步骤执行。

## 什么是技能

技能是一套经过验证的、可重复执行的操作流程。它包含明确的输入、输出、操作步骤和项目特有约束。

## 提炼标准

仅当操作满足**全部**以下条件时才提炼为技能：

1. 在项目中出现 3 次以上的重复模式
2. 有明确的输入和输出
3. 步骤稳定可复现
4. 含有项目特有的约束或规范
5. 能复用到未来的同类任务

**不要创建**：通用且无项目特征的操作（如"读代码"、"修 bug"）

## 目录结构

```
skills/
  <skill-name>/
    SKILL.md    ← 技能定义文件，name 字段必须和目录名一致
```

## YAML frontmatter 要求

```yaml
---
name: skill-name
description: "Use when: 关键词丰富的触发描述"
argument-hint: "输入提示"
---
```
